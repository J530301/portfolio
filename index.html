Personal Project

ShopTime Ecommerce Website

ShopTime is an e-commerce website project built with two connected platforms, one using Laravel and the other using Node.js with React.

The Laravel version handles the core features, including separate authentication for admin and users. Admins can manage products by adding, updating, or deleting them, and also have the ability to edit user accounts. Users can view available products, make purchases, and track their orders.

The second version of ShopTime, built with Node.js and React, offers a different design but provides the same functionalities. Both platforms are part of one project, showcasing the same e-commerce system developed using two different programming stacks.
